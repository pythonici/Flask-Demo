# flask-demo

#### 介绍
Flask和Vue3制作一个单页应用,实现图书信息的增删改查功能

#### 软件架构
    ------flasky  flask代码
      -----venv  虚拟环境
      -----flasky.py 后端代码模块
      -----db_sqlite.py 数据库建表，添加数据用
    ---node_modules  前端依赖模块
    ---data.sqlite  数据库存储文件
    ---src
      -----components  组件目录
      -----FlashMessage.vue  消息模板
      -----HelloWorld.vue   前端模块代码

#### 使用说明

1.  set FLASK_APP=flasky.py
2.  flask run
3.  npm run serve
